# Style Guide: Personal Finance vs Inflation Project

## 📋 Naming Conventions

### Sheet names
- Format: `snake_case`
- All lowercase, words separated by underscores
- Examples: `ipc_indec_raw`, `categories_ipc_mapping`, `analysis`

### Column names
- Format: `snake_case`
- All lowercase, words separated by underscores
- Examples: `total_expenses`, `cpi_monthly_var`, `my_expenses_index`

---

## 🔤 Typography

- Font family: Roboto
- Header font size: 10pt, bold, color `#FFFFFF`
- Data font size: 10pt, regular, color `#434343`
- Chart title: 12pt, bold
- Axis labels: 10pt, regular

---

## 🎨 Table Formatting

### Headers
- Background color: `#1F3864` (dark blue)
- Text color: `#FFFFFF` (white)
- Font: bold
- Alignment: centered
- Font size: 10pt

### Data rows (alternating colors)
- Row 1: `#FFFFFF` (white)
- Row 2: `#DCE6F1` (light blue)
- Text color: `#434343` (dark gray 4)
- Text alignment: left
- Number alignment: right
- Font size: 10pt

### Borders
- Outer border: `#CCCCCC` (medium gray)
- No vertical internal borders
- Light horizontal borders only

---

## 🔢 Number Formats

| Data type | Format |
|---|---|
| Amounts | Number, 2 decimals, thousands separator |
| Percentages | Number, 2 decimals, % symbol |
| Indexes | Number, 2 decimals |
| Dates | MMM YYYY (e.g. May 2024) |

---

## 📊 Chart Style

### Colors
- Personal expenses series: `#1F3864` (dark blue)
- Official CPI series: `#E67E22` (orange)
- Reference line: `#F1C40F` (yellow)

### General
- Title: bold, 12pt
- Legend: visible, positioned at top
- Axis labels: visible on both axes
- Grid lines: light gray
